# WeChat 小学听写练习小程序

微信小程序，支持语文和英语听写，解放家长注意力，让孩子独立完成听写作业。

## 功能特性

- ✅ 支持语文和英语两个科目
- ✅ 按年级/科目/课文选择，支持多选
- ✅ TTS 语音播报（腾讯云 TTS）
- ✅ 语速调节（慢速 0.7x / 正常 1.0x / 快速 1.3x）
- ✅ 音色选择（8种精品音色：女声、男声、童声）
- ✅ 会话持久化（中途退出可继续）
- ✅ 进度显示（实时进度条）
- ✅ 随机顺序播放
- ✅ 听写完成显示词汇列表（英语含释义）

## 技术选型

- **TTS 服务**: 腾讯云 TTS（8种精品音色）
- **本地存储**: `wx.setStorageSync`（语速、音色、进度持久化）
- **词库**: 静态 JSON（语文 + 英语）
- **音频格式**: WAV（16kHz采样率）
- **缓存**: LRU（100词，约500KB）

### 数据格式

**语文词语：**
```javascript
{
  "text": "燕子",
  "pinyin": "yàn zi"
}
```

**英语单词：**
```javascript
{
  "text": "our",
  "pronunciation": "/aʊə(r)/",
  "explain": "我们的"
}
```

> 注：英语单词在听写过程中只显示音标，完成后统一显示单词和释义。

## 快速开始

### 1. 导入项目

1. 打开微信开发者工具
2. 导入项目：选择此目录
3. AppID 选择"测试号"

### 2. 测试功能

1. 选择年级、科目、册次
2. 选择课文（支持多选）
3. 点击"开始听写"
4. 点击播放按钮听词语
5. 点击"下一个"继续

### 3. 配置 TTS

项目已集成腾讯云TTS服务,需要进行配置才能使用语音播放功能:

1. 复制配置文件:
   ```bash
   cp utils/config.example.js utils/config.js
   ```

2. 获取腾讯云API密钥:
   - 访问 https://console.cloud.tencent.com/cam/capi
   - 创建或查看你的SecretId和SecretKey

3. 编辑 `utils/config.js` 填入密钥:
   ```javascript
   module.exports = {
     tencentCloud: {
       secretId: '你的SecretId',
       secretKey: '你的SecretKey'
     }
   }
   ```

4. 开通腾讯云TTS服务:
   - 访问 https://console.cloud.tencent.com/tts
   - 开通服务(可领取免费额度)

详细配置说明请参考: [docs/TTS_CONFIG.md](docs/TTS_CONFIG.md)

## 使用说明

### 选课流程

1. 选择年级（一年级～六年级）
2. 选择科目（语文/英语）
3. 选择册次（上册/下册）
4. 展开课文，勾选词语（可多选）
5. 点击"开始听写"

### 听写流程

1. 进入听写页面后自动播放第一个词
2. 屏幕显示拼音/音标（不显示答案）
3. 学生根据读音写下词语
4. 点击"重播"再听一遍
5. 点击"下一个"进入下一词
6. 完成后显示庆祝页面和所有词汇列表

### 语速调节

在听写页面右上角：
- ⚡ 慢速 (0.7x) - 适合刚开始学习的孩子
- ⚡ 正常 (1.0x) - 标准语速
- ⚡ 快速 (1.3x) - 适合复习巩固

### 音色选择

在听写页面右上角，点击 🎤 选择音色：
- **智瑜**（情感女声）- 默认，适合低年级
- **智云**（通用男声）
- **智希**（通用女声）
- **智梅**（通用女声）
- **智柯**（通用男声）
- **智友**（通用男声）
- **智甜**（女童声）- 适合与孩子同龄
- **智萌**（男童声）- 适合与孩子同龄

> 💡 建议：低年级推荐使用童声或情感女声，高年级可使用通用音色

## 常见问题

**Q: 语音播放失败怎么办？**
A: 检查以下几点：
1. 确认已正确配置腾讯云 API 密钥
2. 确认腾讯云 TTS 服务已开通
3. 检查网络连接是否正常
4. 查看控制台错误信息

**Q: 如何添加更多词库？**
A: 参考 `words/英语-三年级下册.txt` 格式，在 `data/wordDatabase.js` 中添加相应数据即可。

**Q: 音色设置不生效？**
A: 音色设置会自动保存，下次打开会自动恢复。如果仍不生效，请检查腾讯云账户是否有权限使用该音色。
