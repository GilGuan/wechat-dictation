# WeChat 小学听写练习小程序

微信小程序，支持语文和英语听写，解放家长注意力，让孩子独立完成听写作业。

## 功能特性

- ✅ 支持语文和英语两个科目
- ✅ 按年级/科目/课文选择，支持多选
- ✅ TTS 语音播报（腾讯云/讯飞 TTS）
- ✅ 语速调节（慢速/正常/快速）
- ✅ 会话持久化（中途退出可继续）
- ✅ 进度显示
- ✅ 随机顺序播放

## 项目结构

```
wechat-dictation/
├── app.js                 # 主入口
├── app.json               # 小程序配置
├── project.config.json    # 开发工具配置
├── data/
│   └── wordDatabase.js     # 词库数据（语文+英语）
├── pages/
│   ├── lesson-select/     # 选课页面
│   └── dictation/          # 听写页面
```

## 技术选型

- **TTS 服务**: 腾讯云 TTS / 讯飞 TTS
- **本地存储**: `wx.setStorageSync`
- **词库**: 静态 JSON（语文 1-3年级 + 英语 3-4年级）
- **缓存**: LRU（100词，约500KB）

## 词库说明

### 语文（人教版）
- 一年级上/下册
- 二年级上/下册
- 三年级上/下册

### 英语（PEP 人教版）
- 三年级上/下册
- 四年级上/下册

## 快速开始

### 1. 导入项目

1. 打开微信开发者工具
2. 导入项目：选择此目录
3. AppID 选择"测试号"

### 2. 测试功能

1. 选择年级、科目、册次
2. 选择课文（支持多选）
3. 点击"开始听写"
4. 点击播放按钮听词语
5. 点击"下一个"继续

### 3. 配置 TTS（可选）

当前使用测试音频。如需真实词语发音，请参考 `TTS_CONFIG.md` 配置腾讯云或讯飞 TTS。

## 使用说明

### 选课流程

1. 选择年级（一年级～六年级）
2. 选择科目（语文/英语）
3. 选择册次（上册/下册）
4. 选择课文（可多选）
5. 点击"开始听写"

### 听写流程

1. 进入听写页面后自动播放第一个词
2. 听完后自动显示答案（拼音+汉字）
3. 点击"再读一遍"重播
4. 点击"下一个"进入下一词
5. 完成后显示庆祝页面

### 语速调节

在听写页面底部：
- 慢速 (0.7x) - 适合刚开始学习的孩子
- 正常 (1.0x) - 标准语速
- 快速 (1.3x) - 适合复习巩固

## 注意事项

- WeChat 10MB 存储限制
- TTS API 按字符计费
- 多音字需要人工校对
- 词库版权问题需确认

## 开发环境

1. 在小程序后台配置 TTS 服务域名白名单

3. 补充完整词库数据
