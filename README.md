# WeChat 小学听写练习小程序

微信小程序，支持语文和英语听写，解放家长注意力，让孩子独立完成听写作业。

## 功能特性

- ✅ 支持语文和英语两个科目
- ✅ 按年级/科目/课文选择，支持多选
- ✅ **双TTS服务支持**（腾讯云主服务 + 科大讯飞备选服务）
- ✅ **智能主备切换**（主服务故障自动切换到备用服务）
- ✅ 语速调节（慢速 0.7x / 正常 1.0x / 快速 1.3x）
- ✅ 音色选择（8种精品音色：女声、男声、童声）
- ✅ **LRU音频缓存系统**（节省流量和费用）
- ✅ 会话持久化（中途退出可继续）
- ✅ 进度显示（实时进度条）
- ✅ 随机顺序播放
- ✅ 听写完成显示词汇列表（英语含释义）
- ✅ **云函数安全管理密钥**（防止密钥泄露）

## 快速开始

### 1. 导入项目

1. 打开微信开发者工具
2. 导入项目：选择此目录
3. AppID 选择"测试号"

### 2. 开通云开发（必需）

**为什么使用云函数？**
- ✅ **更安全** - 密钥存储在云端，前端代码不可见
- ✅ **易管理** - 随时更新密钥，无需重新发布小程序
- ✅ **防泄露** - 避免密钥被打包到小程序中被反编译

**开通步骤：**

1. 点击工具栏的"云开发"按钮
2. 首次使用会提示开通，按照指引创建云开发环境
3. 记录**环境ID**（例如：`cloud1-xxx`）

### 3. 配置环境变量

在云开发控制台配置环境变量：

**云开发控制台 → 云函数 → 配置 → 环境变量**

```bash
# 腾讯云TTS配置（至少配置一个服务）
TENCENT_SECRET_ID=你的腾讯云SecretId
TENCENT_SECRET_KEY=你的腾讯云SecretKey

# 科大讯飞TTS配置（可选，作为备选服务）
XUNFEI_APP_ID=你的讯飞AppId
XUNFEI_API_KEY=你的讯飞ApiKey
XUNFEI_API_SECRET=你的讯飞ApiSecret
```

**获取密钥：**
- **腾讯云**: https://console.cloud.tencent.com/cam/capi （创建或查看SecretId和SecretKey）
- **科大讯飞**: https://console.xfyun.cn/services/tts （创建应用，获取AppID、API Key、API Secret）

### 4. 配置环境ID

修改 `app.js`，填入你的云开发环境ID：

```javascript
// app.js
App({
  globalData: {
    cloudEnvId: 'your-env-id'  // 👈 改为你的环境ID
  },

  initCloud() {
    wx.cloud.init({
      env: 'your-env-id',  // 👈 改为你的环境ID
      traceUser: true
    })
  }
})
```

### 5. 上传云函数

在微信开发者工具中：

1. 展开左侧 `cloudfunctions` 目录
2. 右键点击 `getTTSConfig` 文件夹
3. 选择"上传并部署：云端安装依赖"
4. 等待上传完成

### 6. 测试功能

1. 选择年级、科目、册次
2. 选择课文（支持多选）
3. 点击"开始听写"
4. 点击播放按钮听词语
5. 点击"下一个"继续

**预期输出：**
```
[云开发] 初始化成功
[听写] 开始初始化TTS服务...
[配置管理器] 从云函数获取配置
[配置管理器] 配置获取成功
[听写] 腾讯云TTS服务已初始化
```

## 技术架构

### 双TTS服务架构

```
用户播放词语
    ↓
优先使用腾讯云（主服务）
    ↓
失败？
    ↓ 是
自动切换科大讯飞（备选服务）
    ↓
成功？
    ↓ 是
更新当前服务
    ↓
继续使用备用服务
```

**优势：**
- ✅ 提升服务可用性
- ✅ 降低单点故障风险
- ✅ 用户无感知切换

### 云函数密钥管理

```
小程序前端
    ↓ 调用云函数
云函数 getTTSConfig
    ↓ 读取环境变量
返回密钥（安全）
    ↓
配置管理器缓存（5分钟）
    ↓
TTS服务初始化
```

**安全性：**
- ✅ 密钥存储在云端环境变量
- ✅ 前端代码不包含密钥
- ✅ 可以随时更新密钥
- ✅ 支持环境隔离

### LRU缓存系统

- **智能缓存Key**: 基于词语+语速+音色+采样率+格式生成
- **LRU淘汰策略**: 综合考虑访问时间和频率
- **容量管理**: 限制8MB / 200个文件
- **自动清理**: 7天未访问自动清理
- **性能提升**: 缓存命中 <50ms（vs 首次播放500-1000ms）

- ✅ 优先使用腾讯云（主服务）

- 腾讯云 `-1` → 讯飞 `3`
- 腾讯云 `0` → 讯飞 `5`
- 腾讯云 `1` → 讯飞 `7`

- **智梅**（通用女声）
- **智柯**（通用男声）
- **智友**（通用男声）
- **智甜**（女童声）- 适合与孩子同龄
- **智萌**（男童声）- 适合与孩子同龄

## 常见问题

**Q: 语音播放失败怎么办？**

检查以下几点：
1. 确认云开发已开通并正确配置
2. 确认云函数 `getTTSConfig` 已上传
3. 确认环境变量已正确设置（在云开发控制台）
4. 检查网络连接是否正常
5. 查看控制台错误信息
6. 如果配置了双服务，系统会自动尝试备用服务

**Q: 为什么要用云函数管理密钥？**

- ✅ **更安全** - 密钥不会被打包到小程序中
- ✅ **易管理** - 随时更新密钥，无需重新发布
- ✅ **防泄露** - 避免密钥被反编译获取
- ✅ **无需本地配置** - 不需要任何配置文件

**Q: 本地配置文件在哪里？**

本项目不使用本地配置文件！所有密钥都通过云函数环境变量管理，配置更加安全。

**Q: 科大讯飞和腾讯云有什么区别？**

- **腾讯云**：主服务，8种精品音色，WAV格式，按字符计费
- **科大讯飞**：备选服务，更多音色可选，MP3格式，免费版每日500次
- 系统会优先使用腾讯云，失败时自动切换到科大讯飞

**Q: 如何添加更多词库？**

参考 `words/英语-三年级下册.txt` 格式，在 `data/wordDatabase.js` 中添加相应数据即可。

**Q: node_modules 这么大，会影响小程序上传吗？**

完全不会。微信开发者工具会自动忽略 `node_modules/` 目录，这些文件不会被打包上传。实际小程序包大小只有约 124KB。

**Q: 云函数调用失败？**

检查以下几点：
1. ✅ 云开发是否已开通
2. ✅ 环境ID是否正确
3. ✅ 云函数是否已上传
4. ✅ 基础库版本 >= 2.2.3

**Q: 控制台报错 "webapi_getwxaasyncsecinfo:fail Failed to fetch"？**

这是**微信开发者工具的已知问题**，不影响云函数正常使用。

- **原因**：开发者工具在初始化云开发时的一个警告
- **影响**：无，云函数依然能正常调用
- **解决**：无需处理，真机上不会出现此错误
- **参考**：https://developers.weixin.qq.com/community/develop/article/doc/00026a5b964580a8bfec0b3b25bc13
