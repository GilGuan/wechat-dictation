# 更新日志

所有重要的更改都将记录在此文件中。

## [3.1.0] - 2026-03-27

### 🔒 重大安全更新
- **使用云函数管理密钥** - 避免密钥泄露风险
- 废弃本地配置文件 - 提升安全性

### 🐛 Bug 修复
- 修复云函数调用逻辑，移除不必要的 errMsg 检查
- 修复 TTS 服务初始化时序问题 - 避免在服务未初始化时调用
- 优化云开发初始化错误处理
- 移除"TTS服务未配置"弹窗提示，改为控制台日志

### 📝 文档更新
- 在 README 添加"webapi_getwxaasyncsecinfo"错误的说明
- 在 app.js 添加云开发初始化已知问题的注释

### ✨ 新增功能

#### 云函数密钥管理
- 新增 `cloudfunctions/getTTSConfig` 云函数
  - 安全地返回TTS服务密钥
  - 使用环境变量存储密钥
  - 支持腾讯云和科大讯飞双服务配置

- 新增 `utils/ttsConfigManager.js` 配置管理器
  - 自动从云函数获取配置
  - 5分钟智能缓存
  - 缓存过期自动刷新
  - 云函数失败时使用过期缓存降级

#### 小程序云开发集成
- `app.js` 新增云开发初始化
  - `initCloud()` 方法
  - `setCloudEnvId()` 方法
  - 云开发环境ID全局配置

#### 双TTS服务支持
- 新增科大讯飞TTS服务 (`utils/xunfei.js`)
  - 支持备选TTS服务
  - 主服务故障时自动切换
  - 免费版每日500次调用

- 优化腾讯云TTS服务
  - 更好的错误处理
  - 自动重试机制

#### 音频缓存系统
- 新增 `utils/audioCache.js` LRU缓存管理器
  - 基于LRU算法的智能缓存
  - 最大缓存100个音频文件
  - 自动清理过期缓存
  - 缓存命中率统计

### 🔧 优化改进

#### 错误处理
- 优化TTS服务错误处理
- 云函数调用失败时优雅降级
- 更详细的控制台日志

#### 配置管理
- 环境变量配置更安全
- 支持云开发控制台在线配置
- 无需本地配置文件

#### 用户体验
- 移除不必要的弹窗提示
- 控制台输出更友好的日志
- 服务初始化完成后才自动播放

### 🗑️ 废弃功能

- ❌ 删除 `utils/config.example.js` - 不再需要本地配置
- ❌ 删除 `docs/TTS_CONFIG.md` - 配置方式已更新

### 📦 依赖更新

- 新增 `wx-server-sdk` 云函数依赖
- 新增 `crypto-js` 用于讯飞鉴权
